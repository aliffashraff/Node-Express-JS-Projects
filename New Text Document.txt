# Back-End Projects using Node.js and Express.js
## Project List
1. Simple Calculator App
2. Rock-Paper-Scissors Game
3. Simple To-Do-List App
4. YouTube Home Page Clone

## Instructions
1. Run 'npm install' in terminal (download node_modules)
2. Create .env file in the root
3. Setup MONGO_URI(Atlas dB connection) in the .env file
4. Run 'npm start' in terminal
